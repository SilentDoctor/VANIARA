# VANIARA - Система Контроля Доступа

Система контроля доступа для сотрудников и посетителей предприятия Геомаш.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![PyQt6](https://img.shields.io/badge/PyQt6-6.11+-green)
![SQLite](https://img.shields.io/badge/SQLite-3-orange)

## Возможности

- **Журнал проходов** - учёт входа/выхода сотрудников
- **Регистрация посетителей** - приход и уход гостей
- **Управление сотрудниками** - добавление, редактирование, поиск
- **Статистика** - анализ посещаемости и опозданий
- **Ролевая система** - разграничение прав доступа

## Установка

### Требования

- Python 3.10 или выше
- Windows 10/11

### Шаги установки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/SilentDoctor/VANIARA.git
cd VANIARA
```

2. **Создайте виртуальное окружение (рекомендуется):**
```bash
python -m venv venv
venv\Scripts\activate
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Тестовые данные

Для демонстрации созданы тестовые пользователи и сотрудники.

### Пользователи системы

| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin123 | Администратор |
| guard1 | guard123 | Охранник |

### Инициализация данных

Если база данных пустая, запустите скрипт инициализации:
```bash
python init_test_data.py
```

Это создаст:
- Администратора (admin / admin123)
- Охранника (guard1 / guard123)
- Тестовых сотрудников

## Роли пользователей

### Охранник
- Просмотр журнала проходов
- Регистрация посетителей (приход/уход)
- Поиск сотрудника по ФИО/табелю
- Отметка опозданий

### Администратор
- Все функции охранника
- Управление сотрудниками (CRUD)
- Управление пользователями системы
- Просмотр статистики проходов
- Экспорт отчётов

## Структура проекта


## Структура базы данных

### users - Пользователи системы
- id, username, password_hash, role, full_name, created_at

### employees - Сотрудники
- id, tab_number, full_name, department, position, photo_path, is_active

### pass_logs - Журнал проходов
- id, employee_id, pass_type, timestamp, guard_id, notes

### visitors - Посетители
- id, full_name, phone, organization, purpose, arrival_time, departure_time, guard_id, employee_visited_id

## Технологии

- **PyQt6** - графический интерфейс
- **SQLite** - встроенная база данных
- **bcrypt** - хеширование паролей
